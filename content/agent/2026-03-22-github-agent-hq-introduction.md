## 核心要点

GitHub 发布 Agent HQ，这是一个将 GitHub 转变为开放生态系统的愿景，让各种 AI 代理在单个平台上统一运行。Anthropic、OpenAI、Google、Cognition、xAI 等公司的编码代理将作为付费 GitHub Copilot 订阅的一部分直接在 GitHub 上可用。

## 技术亮点

### 1. Agent HQ 核心理念

**统一代理平台**
- 不是添加外部工具，而是将代理原生集成到 GitHub 流程中
- 让代理按照你现有的工作方式工作
- 基于 Git、Pull Request、Issues 等你熟悉的 primitives

**多代理协作**
- 编排专业化代理集群执行复杂任务
- 支持并行执行多个代理任务
- 避免使用不连贯的工具或依赖单个代理

### 2. Mission Control：统一指挥中心

**跨平台一致性**
- 跟随你在任何地方工作：GitHub、VS Code、移动端、CLI
- 统一界面分配、监控和管理所有 AI 驱动任务
- 从任何设备跟踪代理进度

**新功能特性**
- 选择代理舰队，并行分配工作
- 细粒度分支控制：控制何时运行 CI 和检查
- 身份功能：控制哪个代理构建任务
- 一键合并冲突解决，改进文件导航和代码注释
- 集成 Slack、Linear、Jira、Teams、Azure Boards、Raycast

### 3. VS Code 中的新功能

**Plan Mode（规划模式）**
- 与 Copilot 配合工作，通过提问帮助你构建分步方法
- 提供前置上下文改善 Copilot 的能力
- 在编写代码前发现差距、缺失决策或项目缺陷

**自定义代理**
- 使用 AGENTS.md 文件创建自定义代理
- 源控制文档设置明确的规则和护栏
- 无需每次重新提示就能塑造 Copilot 的行为

**GitHub MCP Registry**
- VS Code 是唯一支持完整 MCP 规范的编辑器
- 一键发现、安装和启用 MCP 服务器（Stripe、Figma、Sentry 等）
- 为专业任务创建自定义代理
